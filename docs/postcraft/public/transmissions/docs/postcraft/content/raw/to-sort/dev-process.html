Transmissions Dev Process

see also Postcraft Flow : 2024-04-27

For a transmission called example

  1. create a working directory transmissions/src/applications/example (henceforth, example)

  2. sketch out process steps in example/about.md

  3. check against existing services

  4. create any new services that are needed

  5. create example/transmission.ttl

  6. create example/services.ttl

  7. if a manifest is appropriate, create project-dir/manifest.ttl

Running without manifest :

transmissions$ ./run example

With manifest :

./run example project-dir

./run postcraft /home/danny/HKMS/postcraft/danny.ayers.name
