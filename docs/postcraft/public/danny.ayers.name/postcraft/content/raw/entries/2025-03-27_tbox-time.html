Here are the detailed steps for popular XMPP clients:

Using Pidgin/Adium/Gajim:

  1. Configure your XMPP account (if you haven't already): - Username: Create any username (e.g., "user") - Domain: xmpp or localhost - Password: Set any password - Server: localhost (or your Docker host IP) - Port: 5222 - Security: Check "Allow plain text authentication" and "Allow invalid/self-signed certificates"
  2. Join the MUC (Multi-User Chat) room: - Room: shed - Server: conference.xmpp - Nickname: Choose any name

Using a Command Line Client like Profanity:

/connect localhost 5222 username password /join shed@conference.xmpp
