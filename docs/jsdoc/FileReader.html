Class: FileReader

FileReader(config)

FileReader class that extends SourceService. Reads the content of a file and emits a 'message' event with the content and context.

Input

  • data : filename
  • context : sourceFile (if data is not provided)

Output

  • data : file content
  • context : as Input

if context.loadContext is set, that is used as a name in the context for the file content
